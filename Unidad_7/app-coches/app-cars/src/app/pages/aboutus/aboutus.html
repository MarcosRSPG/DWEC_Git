Sobre nosotros

Bienvenido/a a CarStore, una web para explorar coches en venta, ver detalles y publicar tu propio coche de forma sencilla.

¿Qué es este sitio?

CarStore es un proyecto web orientado a practicar un flujo completo: front con Angular + API REST con Node/Express + base de datos MongoDB. El objetivo es ofrecer una experiencia clara, rápida y fácil de usar.

¿Qué puedes hacer aquí?

  • Buscar y filtrar coches por marca/modelo.
  • Ordenar por precio o año.
  • Ver detalles de cada coche.
  • Crear, editar y eliminar anuncios.
